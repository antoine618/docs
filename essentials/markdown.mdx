# Settings & Preferences
Customize Moonrank's settings to match your specific needs and team workflow.
## General Settings
### Website Configuration
- **Primary Domain** - Your main website URL
- **Additional Domains** - Track subdomains or multiple properties
- **Preferred Language** - Language for article generation and UI
- **Timezone** - For accurate scheduling and reporting
### Publishing Options
- **Default CMS** - WordPress, Webflow, manual, or API
- **Auto-Publish** - Automatically publish articles after review
- **Scheduling** - Default day/time for scheduled articles
- **Status** - Post status when publishing (draft, published, scheduled)
## Content Preferences
### Article Generation Settings
- **Default Word Count** - Target length for generated articles
- **Writing Style** - Tone and voice (professional, casual, technical, etc.)
- **Language** - English, Spanish, French, German, etc.
- **Include Sources** - Cite sources in articles
- **Content Angle** - Template or custom approach
### SEO Defaults
- **Meta Description Length** - Target character count
- **Keyword Placement** - Where to include target keywords
- **Internal Link Count** - How many internal links to include
- **External Link Count** - How many external links to include
## Team & Collaboration
### Team Members

### Approval Workflows
- Set up custom approval chains

### Activity Tracking

## Notifications
### Email Alerts
Toggle on/off for:
- Daily ranking updates
- New backlinks detected
- Technical issues found
- Articles published
- Comments on your articles
- Team messages
### Notification Frequency
- **Real-time** - Get alerts immediately
- **Daily Digest** - One email per day with all updates
- **Weekly Summary** - One email per week
- **Off** - Don't send notifications
### Slack Integration
- Connect your Slack workspace
- Get Moonrank notifications in Slack
- Create tasks from Slack messages
- Approve articles from Slack
